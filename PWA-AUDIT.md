# PWA-аудит: Сумма прописью 2.4

- стабильный идентификатор приложения `/`;
- `start_url` и `scope` закреплены за корнем домена;
- добавлены отдельные обычные и maskable-иконки 192/512 px;
- добавлены Apple Web App метатеги и touch icon;
- встроена верхняя панель установки;
- Chrome/Edge используют системный install prompt;
- Safari получает инструкции для «Добавить в Dock» или «На экран Домой»;
- приложение скрывает панель в standalone-режиме;
- Service Worker работает с отдельным кэшем версии 2.4;
- навигация использует network-first, статика — cache-first с обновлением;
- старые кэши удаляются при активации;
- все ключевые файлы доступны офлайн после первого успешного открытия.
