# Обновление до версии 2.3.4

Эта сборка является полной. Не накладывайте ее выборочно поверх предыдущих версий.

1. Удалите старое содержимое корня репозитория, кроме папки `.git`, если работаете локально через Git.
2. Скопируйте в корень репозитория все файлы из архива.
3. Убедитесь, что рядом с `index.html` находятся `CNAME`, `service-worker.js`, папки `assets`, `fonts` и `icons`.
4. Опубликуйте изменения.
5. Первый раз откройте `https://summa.martsynkovskyi.ru/?v=234`.

Сайт использует уникальные файлы `styles-v2.3.4.css` и `app-v2.3.4.js`, поэтому старая закэшированная CSS/JS-связка больше не может подменить новую.
