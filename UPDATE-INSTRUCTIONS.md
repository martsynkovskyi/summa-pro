# Обновление с версии 2.2 до 2.3

Распакуйте содержимое архива поверх текущего репозитория и подтвердите замену файлов.

Папка `fonts` в версии 2.3 не менялась. Оставьте существующую папку `fonts` из версии 2.2 в репозитории.

После публикации один раз откройте сайт с подключением к интернету, чтобы новый Service Worker сохранил файлы для автономной работы.
