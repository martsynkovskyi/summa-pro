# Сумма прописью v2.2

Статический сервис для расчёта НДС и подготовки готовых формулировок суммы прописью для договоров, актов, счетов и иных документов.

Домен: `summa.martsynkovskyi.ru`

## Возможности

- расчёт НДС по ставкам 22% и 5%;
- режимы «НДС включён», «НДС сверху» и «Без НДС»;
- четыре формата записи суммы прописью;
- готовые формулировки «в том числе НДС» и «кроме того НДС»;
- варианты обоснования для операций без НДС;
- копирование любого результата одним нажатием;
- автоматическое склонение рублей и копеек;
- корректный ввод сумм с пробелами, точкой или запятой;
- локальное сохранение черновика в браузере;
- адаптивный интерфейс для телефонов, планшетов и десктопов;
- локальный шрифт Golos Text без CDN и внешних запросов.

## Публикация на GitHub Pages

1. Загрузить содержимое архива в корень репозитория.
2. В настройках репозитория открыть `Settings → Pages`.
3. Выбрать публикацию из нужной ветки и корневой папки.
4. Убедиться, что файл `CNAME` содержит:

```text
summa.martsynkovskyi.ru
```

## Безопасность и приватность

- backend отсутствует;
- внешние библиотеки и CDN не используются;
- аналитика, реклама и трекеры отсутствуют;
- расчёты выполняются исключительно в браузере;
- введённые данные сохраняются только в `localStorage` текущего браузера;
- пользовательские значения выводятся через `textContent`, без вставки в `innerHTML`;
- применяется Content Security Policy без `unsafe-inline`;
- внешние сетевые запросы запрещены через `connect-src 'none'`.

## Шрифт

Интерфейс использует Golos Text, распространяемый по лицензии SIL Open Font License 1.1. Текст лицензии включён в файл `LICENSE-OFL.txt`.
