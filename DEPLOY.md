# Публикация версии 2.3.4

1. Удалите из корня репозитория старые `index.html`, `assets/styles.css`, `assets/app.js` и старые версии этих файлов.
2. Загрузите **всё содержимое этого архива** в корень ветки GitHub Pages.
3. Дождитесь завершения GitHub Actions / Pages deployment.
4. Откройте сайт один раз с параметром `?v=234`, например:
   `https://summa.martsynkovskyi.ru/?v=234`
5. После первого открытия обычный адрес снова можно использовать без параметра.

Ключевые файлы этой сборки имеют новые имена:
- `assets/styles-v2.3.4.css`
- `assets/app-v2.3.4.js`

Поэтому они не могут быть подменены старым кэшем предыдущих версий.
